# E-Commerce Backend System

A scalable and secure backend application for an E-Commerce platform built using Spring Boot and MySQL. The project provides RESTful APIs for user authentication, product management, cart operations, order processing, and payment integration.

---

## Features

- JWT Authentication & Authorization
- Role-Based Access Control (Admin/User)
- Product Management APIs
- Category Management
- Cart & Order Management
- Secure REST APIs using Spring Security
- Razorpay Payment Gateway Integration
- Database Migration using Flyway
- DTO & Mapper Architecture
- Exception Handling
- API Testing with Postman
- Layered Backend Architecture

---
